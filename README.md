# Healthy Feet – Podologische Praxis

Production website for Healthy Feet podology practice with locations in Augsburg and Munich.

## Tech Stack

- **React 18** + TypeScript
- **Vite** (build tool)
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** (animations)
- **React Router** (client-side routing)
- **react-helmet-async** (SEO)

## Development

```bash
npm install
npm run dev     # Start dev server at localhost:8080
npm run build   # Production build
npm run preview # Preview production build
```

## Project Structure

```
src/
├── components/   # Reusable UI components
├── data/         # Centralized data (locations, services, reviews, blog)
├── hooks/        # Custom React hooks
├── lib/          # Utilities (cn, cookies)
├── pages/        # Route page components
└── main.tsx      # App entry point
```
